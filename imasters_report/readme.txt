Widget para exibição do iMasters Report e outros vídeos relacionados diretamente do perfil iMasters no Videolog

== Description ==

Este widget busca sempre o vídeo mais atual do perfil iMasters no Videolog e traz em formato de player para o blog.

== Installation ==

1. Faça o upload do arquivo 'imastersreport.php' para a pasta '/wp-=content/plugins/'
2. Ative o plugin no Menu 'Plugins' do Wordpress.
3. Use e abuse.
