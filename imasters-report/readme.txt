Widget para exibição do iMasters Report e outros vídeos relacionados diretamente do perfil iMasters no YouTube

== Description ==

Este widget busca sempre o vídeo mais atual do perfil iMasters no YouTube e traz em formato de player para o blog.

== Installation ==

1. Faça o upload da pasta 'imasters_report/' para a pasta '/wp-content/plugins/'
2. Ative o plugin no Menu 'Plugins' do Wordpress.
3. Basta acessar o menu widget e posicioná-lo no local desejado.
